List Consumption information for consumed entitlements. Absence of an element in this array for a particular entitlement indicates that the entitlement has not been consumed at all. For sub-user level resource consumption, absence of an element in this array for a particular potential consumer indicates that the entitlement has not be consumed at all by that consumer.
